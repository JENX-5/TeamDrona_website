# Drona Aerospace Website

## Quick Start

```bash
npm install
npm run dev
```

## What's Included

- ✅ Dual video header on home page
- ✅ Interactive terminal asteroid game on home page
- ✅ Team page with JSON data
- ✅ Rocket specifications page
- ✅ Contact page with newsletter signup
- ✅ Smooth animations and particle effects

## Customize Team Data

Edit `/public/data/team.json`:

```json
{
  "name": "Your Name",
  "initials": "YN",
  "role": "Your Role",
  "bio": "Your description",
  "year": "2025",
  "linkedin": "https://linkedin.com/in/yourprofile",
  "email": "your@email.com"
}
```
